    서론

    비어있는 비트맵 만들기 1편: 비어있는 비트맵 만들기

    비어있는 비트맵을 만들었으니 큰 이미지에서 일부만 잘라서 화면에 출력해보겠습니다.
    BitBlt() 함수를 이용하여 이미지를 그리고 그릴 이미지의 범위를 지정해주면 됩니다.

    실습에서 사용된 이미지 파일은 bmp 파일(24 bit)입니다.

    조각 이미지 만들기

    조각 이미지를 만드는 방법은 큰 이미지를 로드한 후 비어있는 비트맵에 그림을 그려주는 것입니다.

    먼저 파일이나 리소스ID를 이용해 큰 이미지(bmpImage)를 로드합니다.
    비어있는 비트맵(sliceBmpImage)을 150x150 사이즈로 만들고
    bmpImage가 가지고 있는 이미지를 sliceBmpImage DC에 복사(BitBlt) 합니다.

    비트맵을 만들 때 주의할 점은 어떤 이미지를 잘라내어 담을지 결정해두고
    해당 이미지와 화소당 비트(BPP)를 똑같이 만들어줘야 합니다.

    DC를 가져올 때에는 BitBlt() 함수는 HDC 타입 데이터를 요구하기 때문에
    CImage 클래스에서 제공하는 GetDC() 함수를 사용합니다.
    GetDC() 함수를 사용했을 경우 반드시 ReleaseDC()를 해줘서 사용한 리소스를 돌려주어야 합니다.

    마지막으로 sliceBmpImage가 가지고 있는 이미지를 화면 DC에다가 그려줍니다.

    OnPaintDC dc(this);

CImage bmpImage;
bmpImage.LoadFromResource(AfxGetInstanceHandle(), IDB_BITMAP1);

CImage sliceBmpImage;ㅖ
sliceBmpImage.Create(150, 150, 24);

// 사용자는 그림에서 250, 250 위치부터 150x150 크기 조각 이미지를 비어있는 비트맵에 0,0 위치에 복사
bmpImage.BitBlt(sliceBmpImage.GetDC(), 0, 0, 150, 150, 250, 250, SRCCOPY);
sliceBmpImage.ReleaseDC();
bmpImage.Detach();

// 사용자는 뷰에서 40, 15가 떨어진 위치에서 150x150 크기의 조각 이미지를 화면에 출력하고 싶습니다.
sliceBmpImage.BitBlt(dc.m_hDC, 40, 15, 150, 150, 0, 0, SRCCOPY);
sliceBmpImage.Detach();

    [F] 큰 이미지에서 조각 이미지 출력하는 함수

    [F] 큰 이미지에서 조각 이미지의 CImage 타입 변수를 반환하는 함수

    정리

    간단히 실습한 과정을 정리해보았습니다.

    1. 조각 이미지 크기를 입력하여 비어있는 비트맵을 만들고 큰 이미지를 일부 복사해야합니다.
      • 비어있는 비트맵과 큰 이미지의 화소당 비트(BPP)는 반드시 동일해야합니다.
    2. 이미지를 복사할 때 복사를 시작할 위치 및 복사할 이미지 크기를 적어야합니다.
    3. BitBlt() 함수를 이용하여 비어있는 비트맵에 큰 이미지를 복사합니다.
      • 복사할 대상DC 매개변수가 HDC를 요구하므로
        • GetDC() 함수를 이용하여 CImage가 가지고 있는 이미지 DC로 접근할 수 있습니다.
        • GetDC() 함수를 사용한 경우 빠르게 ReleaseDC() 함수를 사용하여 사용한 리소스를 되돌려 줍니다.
    4. View에 출력할 조각 이미지를 출력할 위치와 출력할 조각 이미지의 크기를 입력합니다.
    5. 마지막으로 출력된 그림을 보고 흐뭇하게 웃습니다.
