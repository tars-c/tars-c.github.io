    [MFC] CImage : 이미지 정보 가져오기 (크기, BPP, Pixel)

    빠른 링크 (Link!)

    • 서론
    • 이미지 정보 가져오기
      • 가로(폭, Width) 가져오기
      • 세로(높이, Height) 가져오기
      • Pixel당 비트 수(BPP) 가져오기
      • x, y 위치의 픽셀 주소 및 컬러값 가져오기

    서론

    프로그램 개발 시 이미지의 크기 정보 및 가지고 있는 정보가 필요할 때가 있습니다.
    CImage 클래스에서 제공하는 함수에서 이미지 정보를 가져오는 함수를 알아봅니다.

    이미지 정보 가져오기

    가로(폭, Width) 가져오기

    가로 폭 정보를 제공해주는 함수는 GetWidth() 입니다.

    CImage myImage;
myImage.Load(...);
int imgWidth = myImage.GetWidth();

    세로(높이, Height) 가져오기

    세로 높이 정보를 제공해주는 함수는 GetHeight() 입니다.

    CImage myImage;
myImage.Load(...);
int imgHeight = myImage.GetHeight();

    Pixel당 비트 수(BPP) 가져오기

    Pixel당 비트 수 정보를 제공해주는 함수는 GetBPP() 입니다.

    CImage myImage;
myImage.Load(...);
int imgBPP = myImage.GetBPP();

    x, y 위치의 픽셀 주소 및 컬러값 가져오기

    (x, y) 위치의 픽셀 주소값 제공해주는 함수는 GetPixelAddress() 입니다.

    CImage myImage;
myImage.Load(...);
int x = 30;
int y = 30;
int imgBPP = myImage.GetBPP();
COLORREF color;
memcpy(&color, myImage.GetPixelAddress(x, y), imgBPP / 8);
