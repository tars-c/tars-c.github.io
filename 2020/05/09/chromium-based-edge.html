    신문물을 접한 느낌이랄까..


    으아악!!
    산소호흡기를 땔 조짐이 보였던 MS Edge가 크로미움 기반으로 새롭게 변했다는 소식을 접했다.
    출시 날짜를 찾아보니까… 2020년 01월 15일이다. (포스팅이 좀 늦었네..)
    LINK: (MicroSoft) Edge 다운로드 받기

    구닥다리 Edge가 Chromium을 올라타서 새롭게 탄생했다.
    추측이지만 Chromium 기반이라 업데이트도 자동으로 할 것 같다.
    반드시 업데이트 기능을 비활성화 시키도록하자

    주인장은 과거에 Chrome광이었다.
    어느날 Chrome Update를 하다보니 Chrome이 너무 불편해졌다.
    그리고 강제로 Update하는거 너무 싫어한다.
    그 이후로 주인장은 Chrome을 쓰지 않는다.
    Chrome 최신버전 기능을 1도 모르니 이점 참고바란다.
    Chromium 기반으로 만들어진 Vivaldi를 사용하거나 Safari를 사용한다.
    **Chrome은 오래된 크롬 버전만 사용할 뿐이다.**

    다운받아서 얼마나 좋은지 직접 느껴보자!

    • Edge가 지원하는 플랫폼은 다음과 같다.
      • Windows 10
      • Windows 8.1
      • Windows 8
      • Windows 7
      • macOS
      • Android

      • iOS

        (지원 가능한 어마무시한 플랫폼들..)
        아쉽게도 리눅스는 지원하지 않는다.

    설치 화면

    깔끔한 설치화면이다.





    아이콘을 보니까 마이크로소프트 느낌이랑 조금 멀게 느껴졌다.
    (vscode, Visual Studio 느낌이 아닌데?)
    새로운 Edge 브라우저 아이콘을 보니까 태극 느낌도 나고 크롬 느낌도 나는 것 같고…





    인스톨러가 인터넷으로 설치파일을 다운을 받은 후
    백그라운드에서 설치를 진행한다.




    설치가 완료 됐다.
    설치 완료 후 작은 창 하나가 뜨는데 사용자에게 초기화 작업해주는 거랑 비슷하다.
    넘기다 보면 마이크로소프트 계정 로그인으로 데이터를 동기화하여 사용자가 편리하게 사용할 수 있다.



    기능 살펴보기

    일단 메뉴 리스트를 눌러서 보면 크롬이랑 너무 흡사하다. (Chromium 기반이니까 비슷할 수 밖에)
    기본 확대 비율은 125% 로 설정되어 있지만, 110% 사용을 추천한다.



    메모리 사용량 및 속도

    소비되는 메모리를 보기 위해 구버전 크롬과 비교해보았다.
    탭을 두개 사용하고 똑같은 사이트에 접속한 상황이다.

    Edge: 102.6 MB

    Chrome(구버전): 125.2 MB

    소비되는 메모리는 엣지가 더 가볍다.
    (크롬 구버전보다 가벼울 수도 있구나.. 마소 역시 소프트 강자 무섭다 ㄷㄷ)

    크롬 구버전은 약간 버벅임이 있다. (아마 구버전이라 속도면에서 떨어진게 아닐까)
    확실히 엣지를 사용하여 구글을 검색하거나 웹사이트를 서핑할 때 불편함을 못느꼈다.

    하드웨어 가속을 끄고 몇가지 검색도 해보면서 테스트를 다시 해보았다.
    Edge 옵션에서 하드웨어 가속을 꺼도 빠른 느낌이 강했다. (가상머신에서 돌리는건데 이렇게 빨라?)
    구버전 Chrome도 빠르지만 나라면 새로운 Edge가 우세한 느낌이 강했다.
    메모리 사용량 부분은 탭 수가 적고 작업 시간이 짧은 상태로 확인했으므로 평가로는 부적합하니 참고만 하자.



    즐겨찾기 (북마크)

    크롬이랑 비슷한 즐겨찾기다.
    그냥 무난해보인다.




    편리한 설장창

    설정창을 보면 너무나 편리하게 잘 써놔있다.
    자세한건 직접 다운로드를 받아서 살펴보기 바란다.
    크롬 사파리 비발디를 써봤지만 그중에 가장 눈에 확 들어온다.
    125% 확대된 상태에선 설정창 왼쪽 리스트가 안보이니 참고하자.


    가장 마음에 들었던건 웹사이트 추적기능을 잘설명해놨다.
    검색 데이터를 지우는 것도 편리하다.
    진짜 사용자를 고려해서 텍스트부터 위치까지 전부 다 세심하게 맞춰진 것 같아서 좋았다.
    설정창에 비활성화 기능이 유독 많아보이는건 기분탓이겠지 ?




    다운로드 목록

    다운로드된 목록을 봐도 사용자가 쉽게 알 수 있게
    시간도 나오고 파일도 종류별로도 필터링 할 수 있다는 점이다.




    방문 기록 (History)

    방문 기록을 검색 기록이란 이름으로 제공한다. 필터링이 잘될 것 같이 UI가 세심하게 잘짜여졌다. 시간까지 나올 줄은 몰랐다.

    반면에 주인장이 사용하는 최신 Safari는…

    (너무 형편없다)



    닫힌 탭 복원

    닫힌 탭 복원 기능도 상당히 좋다.
    Ctrl+Shift+T 단축키를 이용해서 방금 닫은 탭을 복원할 수 있고
    메뉴 리스트 -> 검색 기록 에 가면
    (단축키 상태가.. 누르기는 불편하다)




    브라우저 데이터 가져오기

    브라우저 데이터 가져오는 기능이 있다.
    크롬과 IE 브라우저에 있는 데이터를 가져올 수 있다.





    앱 (New!)

    앱 기능을 사용해보았다.
    사이트 별로 앱으로 등록할 수 있다.
    이미 등록된 사이트의 앱은 다시 등록할 수 없다.
    윈도우즈 스토어에 있는 앱도 사용이 가능할텐데.. 스토어가 폐허라서 그냥 안쓴다.

    사이트로 등록된 앱을 사용하면 주소창이 없는 작은 브라우저 창이 뜬다.
    주소 창이 없어서 주소 입력은 불가능하다.
    상단바에 번역기도 같이 쓸 수 있어서 나름 쓸모 있어보인다.



    내가 생각하기론 레퍼런스 문서를 자주 참고해야할 때 유용한 기능이라 생각한다.
    (불필요한 리소스 사용은 최소화 하고 필요한 기능만 제공하는 그런 느낌이랄까..)


    등록된 앱은 윈도우즈10에서 아래 그림처럼 시작메뉴(?!)에 등록해서 사용할 수도 있다.




    확장 프로그램

    그냥 크롬에서 사용할 수 있는거 새로운 Edge에서도 사용할 수 있다.



    브라우저 작업관리자

    단축키는 Shift+ESC 이다. 탭이 많은 경우 일일히 탭별로 찾아서 끄는 거 진짜 불편했다.
    크롬과 똑같이 브라우저 내 작업관리자를 지원했다.




    번역

    크롬과 비슷하게 구글이 아닌 빙번역 기능도 제공한다.
    구글과 빙 두 번역간 약간의 차이가 있어 보이긴하다.
    어느 것이 좋은지는 사용자의 선택이다.




    다크모드

    다크모드를 지원하는데 애플에서 말하는 갬성과 똑같이 타이틀바만 지원하고 있다.
    (엣지에서 제공하는 다크모드는 해당 웹사이트에서 설정해야 적용됨)
    제공하지 않아도 다크모드 사용을 원한다면 Chrome DarkReader 확장 프로그램 사용을 추천한다.
    Edge에서 제공하는 다크모드 적용 사진은 이미 이 글에 있으므로 생략한다.




    벤치마크

    JetStream에서 여러 브라우저를 평가한 결과이다.
    우승자는 Edge, 다른 평가에서는 Chrome이 승리하기도 했다.
    파이어폭스한테는 지지 않았다.
    Edge가 크롬을 이기는 것으로 보아 제대로 탈바꿈한건 분명하다.


    브라우저 평가 사이트 보러가기 (이미지 출처)



    결론

    이 밖에도 많은 기능들이 있을 것 같지만 여기까지만 보도록하겠다.
    전반적으로 Chromium 영향을 많이 받아서 크롬과 흡사하다.

    새롭게 탈바꿈 하기전에 Edge 브라우저는 솔직히 쓰레기였다.
    마소에 근무하고 계시는 천재 개발자팀이 힘을 합쳐 새생명을 불어넣어주셨다.
    업데이트를 자주해서 크롬처럼 무거워지는 루트만 타지 않는다면 계속 쓰고싶다.

    확실히 크롬 기반의 엔진을 올라타니까 제대로 탈바꿈했다.
    기능이 많아서 무거울 거라고 생각했지만 그렇게 무겁다는 느낌을 받지 못했다.
    가상머신에서 돌리는데에도 빠르고 부드럽게 돌아간다.

    Chromium, Microsoft 이 두 단어가 값어치를 알 수 있다.
    친구한테 엣지 새롭게 나왔으니 써보라고 추천은 하고 싶다. (추천 못할 정도는 아니다)
    최신 버전 크롬 쓰시는 분들은 큰 메리트를 느끼지 못한다고 한다.

    평가: ★★★★☆
    5점 만점에 4점, 크롬이 별로라면 한번쯤 써볼만하다.
