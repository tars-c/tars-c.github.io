    [MFC][실습] CImage : 이미지 불러오고 출력하기

    빠른 링크 (Link!)

    • 서론

    • 비트맵 파일을 이용하여 비트맵 출력하기
    • [F]리소스ID를 이용해 해당 DC에 이미지를 그려주는 함수
    • [F]이미지 파일을 이용해 해당 DC에 이미지를 그려주는 함수
    • 정리

    서론

    MFC는 이미지를 출력할 때 비트맵 변환과정을 거치게 됩니다.
    과거 비트맵을 출력하기 위해서는 복잡한 과정이 있었지만 시간이 지나면서 많은 변화를 겪게되어
    함수 몇가지를 이용하여 이미지를 아주 간단하게 출력할 수 있습니다.

    비트맵(*.bmp) 확장명을 가진 파일을 불러오고 간단히 출력해보도록 하겠습니다.
    OnPaint에서 작업을 했기에 dc 변수의 타입은 OnPaintDC입니다.

    이미지 크기가 큰 경우에는 초기화되는 시점에 미리 로드하여
    멤버 및 전역변수로 접근하여 사용하는 것이 좋습니다.
    이 예제에서는 아주 간단하게 알아보는 것이므로 따로 작업하지 않습니다.

    이미지를 단순히 불러오고 화면에 출력하는건 CImage 클래스가 적합합니다.
    CImage 클래스는 #include <atlimage.h>를 필요로 합니다.

    비트맵을 로드 하는 방법은 총 2가지가 존재합니다.

    리소스ID를 이용하여 비트맵 출력하기

    LoadFromResource() 함수를 이용하려면 Resource View에서 비트맵 파일을 추가합니다.
    추가한 비트맵 파일에 리소스ID를 변경 및 부여하도록 합니다.
    비트맵을 불러오려면 CImage 클래스에서 제공하는 LoadFromResource() 함수를 이용합니다.

    CImage bmpImage;

bmpImage.LoadFromResource(AfxGetInstanceHandle(), IDB_BITMAP1);
bmpImage.BitBlt(dc.m_hDC, 400, 15, 300, 300, 0, 0, SRCCOPY);
//bmpImage.Detach();

    비트맵 파일을 이용하여 비트맵 출력하기

    비트맵 파일을 불러올 땐 CImage 클래스가 제공하는 Load() 함수를 이용합니다.
    Load() 함수는 JPEG, GIF, BMP 및 PNG 확장명을 가진 이미지 파일을 불러옵니다.

    비트맵을 화면에 출력하기 전에 파일을 제대로 불러왔는지 여부를 판단해야합니다.
    Load() 함수가 반환하는 타입은 H_REUSLT 입니다.
    반환값은 S_OK(성공) 및 S_FALSE(실패) 값을 가지고 있습니다.
    SUCCEEDED, FAILED 매크로 함수를 이용하여 성공 및 실패 여부 확인을 조건문에 활용할 수 있습니다.

    파일을 불러올 때 기본위치는 cpp 소스파일이 포함된 프로젝트 위치로 지정되어 있습니다.
    상대경로를 사용하여 이미지 폴더에 있는 이미지 파일을 불러오면 됩니다.

    CImage 인스턴스에서 제공되는 BitBlt() 함수를 이용하면
    특정 DC에 CImage 인스턴스가 가지고 있는 이미지를 복사 및 출력할 수 있습니다.
    BitBlt() 함수는 ATL 함수이기에 DC는 HDC 타입을 요구합니다.

    CImage bmpImage;
if (FAILED(bmpImage.Load(_T(".\\res\\img\\RAY.bmp"))))
{
	return;
}
bmpImage.BitBlt(dc.m_hDC, 80, 15, 300, 300, 0, 0, SRCCOPY);
// bmpImage.Detach();

    간단히 위에 사용 된 BitBlt() 함수를 해석해보면
    (0,0)에서부터 전체크기만큼 bmpImage 인스턴스가 가지고 있는 이미지 데이터를
    dc.m_hDC에다가 (80,15)위치에서 (300,300)크기만큼의 이미지를 복사해라 정도가 되겠네요?!

    [F]이미지 파일을 이용해 해당 DC에 이미지를 그려주는 함수

    정리

    비트맵 확장명을 가진 파일을 리소스ID를 이용 및 파일을 직접 불러오고
    이미지를 출력하는 방법에 대해 알아보았습니다.

    1. 이미지를 간단히 불러오고 화면에 출력하는데는 CImage 클래스가 적합합니다.
    2. LoadFromResource() 함수는 리소스 ID를 이용해 비트맵을 불러옵니다.
    3. Load() 함수는 이미지 파일을 이용해 비트맵을 불러옵니다.
