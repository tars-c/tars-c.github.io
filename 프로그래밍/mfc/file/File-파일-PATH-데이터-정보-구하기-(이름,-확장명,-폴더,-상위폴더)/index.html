    [MFC] File : 파일 PATH 데이터로 정보 구하기 (이름, 확장명, 폴더, 상위폴더)

    간단히 함수를 만들어보았습니다.
    아래 함수들은 파일 PATH 정보를 매개변수로 하여 사용해야 합니다.

    • 파일 PATH 정보 예시:
      • C:\Temp\Text1.txt
      • C:\Windows\notepad.exe
    • 오타 및 잘못된 내용에 대해 피드백 해주시는건 언제든 환영합니다.
