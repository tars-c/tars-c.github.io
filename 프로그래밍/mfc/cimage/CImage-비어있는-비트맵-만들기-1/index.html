    비어있는 비트맵 만들기 2편: 화면에 조각 이미지 출력하기

    이미지를 편집할 때 일부 이미지를 잘라야할 때가 있습니다.
    MFC에서는 이미지를 자르기 위해서는 비어있는 비트맵을 만들어야합니다.
    실습에서 사용된 이미지 파일은 bmp 파일(24 bit)입니다.

    이 글에서는 비어있는 비트맵을 만드는 방법에 대해 적어보겠습니다.

    비어있는 비트맵 생성

    비어있는 비트맵을 만드는 방법은 CImage 클래스에서 제공하는 Create() 함수를 사용합니다.
    비어있는 비트맵을 만들고 BitBlt() 함수를 사용하여 DC에 그림을 그려줄 수 있습니다.
    비어있는 비트맵이 너무 작은 경우 검은 배경이 나타날 수 있으니 크기에 주의해야합니다.

    • 기본적으로 비어있는 비트맵을 만들 때 필요한 요소는 3가지입니다.
      1. 비트맵 가로 크기(Wdith)
      2. 비트맵 세로 크기(Height)
      3. 비트맵 화소 당 비트(BPP)
    OnPaintDC dc;(this);
sliceBmpImage.Create(150, 150, 24);
//sliceBmpImage.BitBlt(dc.m_hDC, 40, 15, 300, 300, 0, 0, SRCCOPY);

    150x150 크기의 24bit 비어있는 비트맵을 생성합니다.
    비트맵을 처음 만들게 되면 검은색으로 칠해져 있습니다. (비트맵 색상값이 0으로 채워져 있음)
